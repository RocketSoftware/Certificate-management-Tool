To run this tool execute pfxmenu.bat
For help execute pfxhelp.bat

This tool depends on the following products: 
Windows powershell 
Jave Runtime Engine (JRE)
openssl

The above tools must be defined in your session path. Modify scertmgr.cmd to reflect the location
of your openssl and JRE dependent environment variables.
For powershell, consult https://msdn.microsoft.com/en-us/powershell/scripting/setup/installing-windows-powershell
For openssl, a minimum of version 1.0.1m is required from openssl.org
For JRE, a version of 1.8 is required from oracle.com.
