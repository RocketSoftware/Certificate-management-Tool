To run this tool execute pfxmenu.bat
For help execute pfxhelp.bat
This tools depends on powershell, JVM, and openssl and defined in your session path
For powershell, consult https://msdn.microsoft.com/en-us/powershell/scripting/setup/installing-windows-powershell
For openssl, a minimum of version 1.0.1m is required from openssl.org
For JVM, a version of 1.8 is required from oracle.com.
